Module Ann.Num

val pow : int -> int -> int

pow x y is x raised to the yth power. Exponentiation for ints.

val bin_pow : int -> bool

bin_pow x is true if x is an int that is a power of 2, else false.

val pad : int -> int

pad n is the next closest larger binary power. If bin_pow n is true then just returns n.
